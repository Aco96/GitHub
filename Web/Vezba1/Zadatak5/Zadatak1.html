Web citac

Web citac je aplikacija koja je namenjena prikazu sadrzaja web stranica.
Stranice mogu dobaviti kako iz fajl-sistema,tako i putem TCP/IP mreze.
Protokol kojim komunucuraju web citac i web server je HTTP protokol. Danas su najpoznatiji sledeci web citaci:

  2. Mozila
  3. Opera
